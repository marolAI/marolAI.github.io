Evaluating Modern Language-Only Baseline (BERT) for Visual Question Answering (VQA)

VQA involves answering questions about images using natural language. Initially, RNNs, LSTMs, and GRUs with pretrained word embeddings were used for question-only baselines. However, with advancements, large-scale pretrained language models like BERT have become involved. This project aims to retrain and evaluate a language-only baseline VQA model using BERT.

Robust Machine Learning With Imbalanced Data: Expanding Mixup

Imbalanced learning focuses on learning from imbalanced data. SMOTE is a technique to address this issue, but it has limitations in predicting the minority class when data is highly skewed or outside the training set. This is because of the use of ERM for training. Mixup is an algorithm that can overcome this limitation. The project aims to combine SMOTE and Mixup to predict fraudulent credit card transactions in an imbalanced dataset.

Smart Identity: classify and extract personal information from the ECOWAS ID card.

Processing paper-based documents is a significant industry problem. It not only demands human resources but also consumes time, leading to frustration and inefficiency. This project aims to automate the extraction of personal information from ECOWAS ID cards using a model based on CNN for classification and Pytesseract for OCR.
